# Perfect World — Отметить подарки / Mark Gifts

---

## Русский

Chrome-расширение, которое автоматически отмечает все чекбоксы на странице подарков [pwonline.ru](https://pwonline.ru/promo_items.php).

### Установка

1. Нажми **Code → Download ZIP** на этой странице и распакуй архив
2. Открой Chrome и перейди на `chrome://extensions`
3. Включи **Режим разработчика** (переключатель в правом верхнем углу)
4. Нажми **Загрузить распакованное расширение** и выбери распакованную папку
5. Готово!

### Использование

Просто открой страницу подарков на pwonline.ru — все чекбоксы отметятся сами.

## English

A Chrome extension that automatically checks all checkboxes on the gifts page of [pwonline.ru](https://pwonline.ru/promo_items.php).

### Installation

1. Click **Code → Download ZIP** on this page and unzip the archive
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer mode** (toggle in the top right corner)
4. Click **Load unpacked** and select the unzipped folder
5. Done!

### Usage

Just open the gifts page on pwonline.ru — all checkboxes will be checked automatically.
